Conception: A term-expansion service for news topics

Example Calls

  • https://conception.global-resonance.com/most_similar?positive=king&positive=woman&negative=man&topn=20
  • https://conception.global-resonance.com/n_similarity?ws1=Sushi&ws1=Shop&ws2=Japanese&ws2=Restaurant
  • https://conception.global-resonance.com/similarity?w1=Sushi&w2=Japanese
  • https://conception.global-resonance.com/model?word=restaurant

Notes

  • Requests for unrecognised terms will return a 500 Internal Server Error
  • Terms are drawn from Google's word2vec result set produced Sept2014
  • The 'model' call returns a base64 encoded version of the word
  • Code maintained at: https://github.com/Chaffleson/word2vec-api
